OpenLINA
========

OpenLINA is an MIT-licensed open source project for managing AI coding assistants from a local interface.

L.I.N.A. stands for:

- Local
- Interface
- Node-based
- Automation

Expanded, that becomes:

Local Interface for Node-based Automation

The name is technical, but it is also personal: LINA is the dog behind the project name and logo.

What OpenLINA is
----------------

OpenLINA is a local GUI for managing AI coding assistants such as Codex, Claude Code, and similar agent-driven developer tools.

Its goal is to provide:

- a central interface for multiple assistants
- a local-first workflow
- a Node.js-based foundation
- an open source platform the community can extend

Development
-----------

Requirements:
- Node.js >= 22.5.0
- npm

Run locally:
- npm install
- npm run dev

Build for production:
- npm run build
- npm start
