# OpenLina

OpenLina is an MIT-licensed open source project for managing AI coding assistants from a local interface.

The name `L.I.N.A.` stands for **Local Interface for Node-based Automation**.

It is technical by design:

- `L` = `Local`: the project is meant to run on your machine and keep control close to the developer.
- `I` = `Interface`: it provides a usable GUI for working with coding assistants instead of raw terminal-only flows.
- `N` = `Node-based`: the platform is built around a modern Node.js stack.
- `A` = `Automation`: the goal is to coordinate and streamline AI-assisted development work.

It is also personal: **LINA is the dog behind the name and the logo**.

## What OpenLina Is

OpenLina is a local GUI for managing AI coding assistants such as Codex, Claude Code, and other agent-style developer tools from one place.

The project is intended to make local AI-assisted development more practical by giving you:

- a central interface for multiple assistants
- a local-first workflow
- a developer-oriented runtime based on Node.js and Next.js
- an open source base that can be extended by the community

## Development

### Requirements

- Node.js `>= 22.5.0`
- npm

### Run Locally

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
npm start
```

## Status

OpenLina is in active early development.

The goal is simple: build a strong local control surface for AI coding workflows, keep it open, and make it useful enough that other developers can adapt it to their own stack.
